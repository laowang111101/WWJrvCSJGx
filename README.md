# 网上摄影工作室管理系统 java1056

## 项目概述

网上摄影工作室管理系统是一款基于前后端分离架构的在线平台，旨在提供摄影作品的分类管理、作品展示、社区互动以及用户管理等多样化功能。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：Idea, vscode
- 依赖框架：mybatis-plus, element

## 功能模块

### 系统角色

- 用户
- 超级管理员

### 核心功能

- **作品分类管理**：对摄影作品进行分类管理，便于浏览与检索。
- **摄影作品管理**：上传、编辑和展示摄影作品。
- **摄影圈管理**：构建摄影爱好者社区，促进用户间的互动交流。
- **公告管理**：发布和管理系统公告，确保信息的及时传达。
- **用户管理**：对用户信息进行管理，维护平台秩序。
- **个人中心**：用户可查看和编辑个人信息，管理个人作品等。
- **个人后台**：为用户提供管理后台，进行个性化设置。

## 运行环境

- 后端：Java 1.8+
- 前端：现代浏览器
- 数据库：MySQL 5.7+

## 部署步骤

1. 环境准备：确保JDK、MySQL等运行环境已正确安装。
2. 数据库部署：导入MySQL数据库脚本。
3. 后端部署：编译后端代码，部署到服务器。
4. 前端部署：编译前端代码，部署到Web服务器。
